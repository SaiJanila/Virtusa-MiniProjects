**Library Management System (Java)**

**Project Description**

This project is a console-based Library Management System developed using Core Java.
It helps manage books, track issued and returned books, and calculate fines for late returns.

**Objective**

To automate basic library operations such as adding books, issuing, returning, and searching books using Java.

**Features**

Add new books to the system,  
Issue books with issue date,  
Return books with return date,  
Fine calculation for late returns (after 7 days),  
Remove books (only if not issued),  
Search books by title,  
Display all available and issued books

**Concepts Used**

Object-Oriented Programming (OOP),  
Classes and Objects,  
ArrayList (dynamic storage),  
Loops and Conditional Statements,  
Exception Handling (try-catch),  
Date Handling (LocalDate),  
User Input using Scanner

**Technologies Used**

Java (Core Java),  
VS Code
